# 组件内容区模板

这个目录保存可直接复用的组件内容区模板。每个模板都是一个独立 `.vue` 文件，只包含可填入 `3 组件区` 槽位的组件内部内容。

使用规则：
- 组件内容区模板只进入分块布局模板的 `3 组件区` 槽位。
- 标题区、胶囊按钮区、辅助信息区、单位区、说明区都配置在父级分块布局模板中。
- 组件内容区模板自身可显示一个顶部标题条：高度 `20px`，标题水平居中，上边距 `3px`。
- 需要移除标题时传入 `showContentTitle: false`；父级分块布局模板只有一个槽位时必须隐藏该标题。
- 组件内容区模板外观统一为圆角矩形，不绘制边框线；空槽位的虚线提示属于分块布局模板，不属于组件内容区模板。
- 没有合适模板时，使用 ECharts 自开发新的独立 Vue 组件内容区模板，再注册到模板库。
